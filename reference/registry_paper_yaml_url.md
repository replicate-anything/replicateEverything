# GitHub raw URL for a registry paper stub yaml

GitHub raw URL for a registry paper stub yaml

## Usage

``` r
registry_paper_yaml_url(
  folder,
  registry_repo = DEFAULT_REGISTRY_REPO,
  ref = "main"
)
```

## Arguments

- folder:

  Registry folder name.

- registry_repo:

  Registry repository slug.

- ref:

  Git ref.
