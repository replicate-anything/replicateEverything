# Check whether a precomputed artifact is available

Check whether a precomputed artifact is available

## Usage

``` r
artifact_available(doi, what, repo = NULL)
```

## Arguments

- doi:

  Character. DOI of the paper.

- what:

  Character. Replication identifier (e.g., `"fig_1"`).

- repo:

  Optional repository slug.

## Value

Logical scalar.
