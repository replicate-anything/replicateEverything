# Fill study-repo fields using a materialized local `replication.yml`

Fill study-repo fields using a materialized local `replication.yml`

## Usage

``` r
complete_folder_study_meta(meta, study_root = NULL)
```

## Arguments

- meta:

  Parsed metadata passed to Stata dependency helpers.

- study_root:

  Local study repository root.

## Value

Updated `meta` list.
