# Relative artifact paths to try under registry `papers/<folder>/`

Relative artifact paths to try under registry `papers/<folder>/`

## Usage

``` r
registry_artifact_rel_paths(what, rep = NULL, ctx = NULL)
```

## Arguments

- what:

  Replication id.

- rep:

  Optional replication entry.

- ctx:

  Optional paper context (for `manifest.json`).
