# Candidate keys for `replicateEverything.study_folders` lookups

Candidate keys for `replicateEverything.study_folders` lookups

## Usage

``` r
study_folder_map_keys(meta, ctx = NULL)
```

## Arguments

- meta:

  Parsed replication.yml contents.

- ctx:

  Paper context from
  [`paper_context()`](https://replicate-anything.github.io/replicateEverything/reference/paper_context.md).

## Value

Character vector of non-empty keys (no duplicates).
