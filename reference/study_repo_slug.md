# Resolve GitHub repo slug for a folder-backed study

Resolve GitHub repo slug for a folder-backed study

## Usage

``` r
study_repo_slug(meta, ctx = NULL)
```

## Arguments

- meta:

  Parsed replication.yml contents.

- ctx:

  Paper context from
  [`paper_context()`](https://replicate-anything.github.io/replicateEverything/reference/paper_context.md).

## Value

Character repo slug.
