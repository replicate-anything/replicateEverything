# Read study package source from GitHub (no install required)

Read study package source from GitHub (no install required)

## Usage

``` r
read_package_repo_source(name, repo, ref = "main")
```

## Arguments

- name:

  Base name without `.R` (e.g. `make_figure_1`).

- repo:

  GitHub slug.

- ref:

  Git ref.

## Value

Character lines or empty.
