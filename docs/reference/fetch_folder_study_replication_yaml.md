# Fetch replication metadata from a folder-backed study repository

Fetch replication metadata from a folder-backed study repository

## Usage

``` r
fetch_folder_study_replication_yaml(meta, ctx = NULL)
```

## Arguments

- meta:

  Parsed registry stub.

- ctx:

  Paper context from
  [`paper_context()`](https://replicate-anything.github.io/replicateEverything/reference/paper_context.md).

## Value

Parsed yaml list or `NULL`.
