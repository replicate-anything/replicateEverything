# Whether a prep step output already exists

Whether a prep step output already exists

## Usage

``` r
prep_output_ready(prep, ctx, meta = NULL)
```

## Arguments

- prep:

  Prep entry from `replication.yml`.

- ctx:

  Paper context from
  [`paper_context()`](https://replicate-anything.github.io/replicateEverything/reference/paper_context.md).

- meta:

  Optional parsed replication metadata.
