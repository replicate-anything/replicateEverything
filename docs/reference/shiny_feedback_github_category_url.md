# Build a category-only GitHub new-issue URL (no user body)

Build a category-only GitHub new-issue URL (no user body)

## Usage

``` r
shiny_feedback_github_category_url(
  category,
  repo = "replicate-anything/replicateEverything"
)

shiny_feedback_category_url(
  category,
  repo = "replicate-anything/replicateEverything"
)
```

## Arguments

- category:

  Allowlisted category.

- repo:

  GitHub `owner/repo` slug.

## Value

Character URL.
