# User-facing message for installing a study replication package

User-facing message for installing a study replication package

## Usage

``` r
study_package_install_message(meta, ctx)
```

## Arguments

- meta:

  Parsed replication metadata (registry stub or package yaml).

- ctx:

  Paper context from
  [`paper_context()`](https://replicate-anything.github.io/replicateEverything/reference/paper_context.md).

## Value

Character string.
