# Extract the surname from a single author name

Handles middle initials (e.g. “Robert A. Blair” \\\rightarrow\\ Blair)
and compound surnames (e.g. “Andrés Vargas Castillo” \\\rightarrow\\
Vargas Castillo).

## Usage

``` r
first_author_surname(name)
```

## Arguments

- name:

  Character scalar.

## Value

Character scalar.
