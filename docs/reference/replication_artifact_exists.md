# Check whether a replication artifact exists

Check whether a replication artifact exists

## Usage

``` r
replication_artifact_exists(
  rep,
  doi,
  folder = NULL,
  repo = NULL,
  study_root = NULL
)
```

## Arguments

- rep:

  Replication entry list.

- doi:

  Paper DOI or lookup key.

- folder:

  Optional registry folder name.

- repo:

  Optional repository slug.

- study_root:

  Optional local study root for direct file checks.
