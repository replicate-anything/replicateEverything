# Maintainer build function name for a study kind

Maintainer build function name for a study kind

## Usage

``` r
study_build_function(kind)
```

## Arguments

- kind:

  Output of
  [`replication_kind()`](https://replicate-anything.github.io/replicateEverything/reference/replication_kind.md).

## Value

`"build_study_outputs"`.
