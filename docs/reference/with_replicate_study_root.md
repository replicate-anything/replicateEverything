# Run an expression with `REPLICATE_STUDY_ROOT` set for folder studies

Run an expression with `REPLICATE_STUDY_ROOT` set for folder studies

## Usage

``` r
with_replicate_study_root(study_root, expr)
```

## Arguments

- study_root:

  Normalized study repository path, or `NULL` to skip.

- expr:

  Expression to evaluate.

## Value

Value of `expr`.
